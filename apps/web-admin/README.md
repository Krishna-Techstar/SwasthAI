# SwasthAI Web Admin

Next.js admin console for SwasthAI operations. The app is implemented with JavaScript and JSX only.

## Run

```bash
pnpm --filter web-admin dev
```

Open http://localhost:3000/dashboard.

## Available Routes

- `/dashboard`
- `/doctors`
- `/nurses`
- `/patients`
- `/ai-diagnostics`
- `/appointments`
- `/billing`
- `/analytics`
- `/verification`
- `/notifications`
- `/support`
- `/audit-logs`
- `/settings`
